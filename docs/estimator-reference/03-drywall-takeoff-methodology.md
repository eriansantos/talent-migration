## 1. Wall Measurement (SF of Drywall)

### Fórmula
```
Wall Area = Length (LF) × Height (FT)
```

### Processo
1. Medir linear feet (LF) de cada parede
2. Multiplicar pela altura da parede
3. Deduzir aberturas grandes (project dependent)

### Exemplo
| Item | Valor |
|---|---|
| Wall Length | 100 LF |
| Wall Height | 10 FT |
| Gross Area | 1,000 SF |

### Openings Deduction Rule

| Opening Type | Deduct? |
|---|---|
| Doors (3x7) | Usually NOT deducted |
| Windows | Deduct if > 15–20 SF |
| Large openings | Always deduct |

---

## 2. Ceiling Measurement (SF of Drywall)

### Fórmula
```
Ceiling Area = Length × Width
```

### Exemplo
| Item | Valor |
|---|---|
| Room Size | 20' × 30' |
| Ceiling Area | 600 SF |

---

## 3. Double-Sided Walls (IMPORTANTE)

> Most interior partitions get drywall on both sides.

```
Total Drywall SF = Wall Area × 2
```

Exemplo: 1,000 SF one-side → 2,000 SF both sides

---

## 4. Height Considerations

| Condição | Impacto |
|---|---|
| Walls over 10' | Produção cai (aumento de custo) |
| Multi-height ceilings | Break out separadamente |
| Soffits / bulkheads | Medir separadamente |

---

## 5. Waste Factor

| Project Type | Waste % |
|---|---|
| Simple layout | **5%** |
| Medium complexity | **8–10%** |
| High cuts/angles | **12–15%** |

---

## 6. Additional Items (Often Missed)

Incluir no takeoff:
- Soffits / bulkheads
- Shaft walls
- Furring walls
- Ceiling drops
- Returns at openings
- Columns wrapped in drywall

---

## 7. Quick Estimating Shortcut (Conceptual)

Para orçamento preliminar:
- **Walls:** ~ 8–12 SF de drywall por SF de floor area (both sides incluídos)
- **Ceilings:** = 1.0 × floor area

---

## Key Estimating Risks

- Não contabilizar os dois lados das partitions
- Esquecer paredes altas ou stairwells
- Ignorar access constraints (lifts/scaffolding)
- Não separar rated vs non-rated assemblies
- Esquecer teto com framing vs ACT grid

## Perguntas que o estimator faz para precificar drywall

1. Project type (commercial, residential, public)
2. Wall heights
3. Stud spacing/type
4. Level of finish (Level 4, 5, etc.)
5. Location (for labor rates)
